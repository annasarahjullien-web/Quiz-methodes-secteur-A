# Quiz Secteur A SVTU - Méthodes de Laboratoire

Quiz complet avec **20 questions sur les vraies méthodes du secteur A** de l'agrégation SVTU.

## Comment utiliser?

### Option 1: Tester localement (30 secondes)
1. Téléchargez le fichier `index.html`
2. Double-cliquez dessus
3. Le quiz s'ouvre dans votre navigateur ✅

### Option 2: GitHub Pages (5 minutes)
1. Créez un repo GitHub public: `quiz-secteur-a`
2. Uploadez `index.html`
3. Settings → Pages → Branch main → Save
4. Attendez 1-2 minutes
5. Accédez à: `https://votre-username.github.io/quiz-secteur-a`
